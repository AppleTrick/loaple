## react + typescript 초기 세팅

```
yarn add typescript @types/node @types/react @types/react-dom @types/jest
```

## react-redux 설치

```
yarn add redux react-redux @types/react-redux
yarn add @reduxjs/toolkit
```

```
npm install -D eslint
```
